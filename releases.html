Release notes

What's new in Cats Lock. Get the latest version on the Mac App Store.

v1.1.0

May 19, 2026

New

  • Global lock shortcut. The unlock shortcut now fires from any app on the system. No need to bring Cats Lock to the foreground first.
  • Customizable shortcut. Record your own combo in Settings. The same shortcut drives both the global hotkey and the in-process unlock.
  • Hide from Dock. Optional menu-bar-only mode. No Dock icon, no Cmd+Tab presence. The menu bar icon and the main window stay accessible.
  • Prevent sleep while locked. Holds the display awake while the keyboard is locked so the overlay stays visible if you step away.

Improvements

  • First-time Stealth Lock now shows a quick confirmation so you know what to expect. Subsequent toggles apply silently.
  • Caps Lock no longer blocks unlock.
  • The warning sound stays quiet while you hold the unlock combo's modifier keys.
  • Launch at Login now boots quietly into the menu bar instead of opening the main window on every login.
  • Onboarding updated to introduce Stealth Lock, the unified unlock shortcut, and sleep prevention.

Under the hood

  • Main window rebuilt on AppKit so it can pull cleanly to the foreground in menu-bar-only mode.
  • Hardened Runtime enabled.
  • No new entitlements. Still fully sandboxed. Still no Accessibility prompt.

v1.0.0

May 15, 2026

New

  • Initial public release on the Mac App Store.
  • Lock the keyboard with ⌘L from the main window or the menu bar.
  • Friendly overlay while locked, with customizable warning sound on each blocked keypress.
  • Stealth Lock mode for silent, invisible locking during meetings or recordings.
  • Launch at Login support.
  • Fully sandboxed. No Accessibility prompt. No account. No analytics.
